# Active Directory & Group Policy

## Objective

Build and manage an enterprise-style Active Directory environment using Windows Server 2022 to simulate real-world IT administration, user management, Group Policy configuration, and domain services.

This phase focused on domain setup, user administration, drive mapping, remote access, and centralized management through Group Policy Objects (GPOs).

---

## Environment

| System | Role | IP Address |
|---------|------|-------------|
| Windows Server 2022 | Domain Controller / DNS | 192.168.10.10 |
| Windows 11 | Domain Joined Client | 192.168.10.20 |

---

## Tasks Performed

- Configured Active Directory Domain Services (AD DS)
- Joined Windows 11 client to the domain
- Created and managed domain users and groups
- Configured Organizational Units (OU)
- Implemented Group Policy Objects (GPO)
- Configured shared drive mapping
- Tested remote connectivity and domain communication
- Troubleshot domain join and connectivity issues

---

## Key Skills Demonstrated

- Active Directory Administration
- Group Policy Management
- Domain Management
- User & Group Administration
- Shared Drive Mapping
- DNS Troubleshooting
- Windows Server Administration
