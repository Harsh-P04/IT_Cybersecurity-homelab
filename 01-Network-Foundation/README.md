# Network Foundation

## Objective

Build and configure an enterprise-style virtual network environment to support IT administration, cybersecurity monitoring, and endpoint communication.

This phase focused on establishing stable networking between virtual machines, validating connectivity, enabling remote access, and configuring shared resources required for later Active Directory and cybersecurity activities.

---

## Environment

| System | Role | IP Address |
|---------|------|-------------|
| Windows Server 2022 | Domain Controller / DNS Server | 192.168.10.10 |
| Windows 11 | Domain Client | 192.168.10.20 |
| Ubuntu Server | Helpdesk + SIEM | 192.168.10.30 |
| Kali Linux | Security Testing | 192.168.10.40 |

---

## Tasks Performed

- Installed and configured virtual machines using VirtualBox
- Configured bridged/internal networking between endpoints
- Assigned and validated IP addressing
- Verified endpoint communication using ping and network troubleshooting
- Enabled remote connectivity between systems
- Configured shared drive access
- Troubleshot connectivity and firewall-related issues
