# Helpdesk Ticketing System (osTicket)

## Objective

Deploy and configure an enterprise-style helpdesk ticketing system using osTicket on Ubuntu Server to simulate real-world IT support operations and ticket resolution workflows.

This phase focused on Linux server administration, Apache and MariaDB configuration, user support scenarios, and troubleshooting common IT incidents.

---

## Environment

| System | Role | IP Address |
|---------|------|-------------|
| Ubuntu Server | osTicket Server | 192.168.10.30 |

---

## Technologies Used

- Ubuntu Server
- Apache2
- MariaDB
- PHP
- osTicket

---

## Tasks Performed

- Installed and configured Apache2 web server
- Configured MariaDB database
- Installed and deployed osTicket
- Configured ticket workflows
- Simulated IT support scenarios
- Managed helpdesk operations
- Troubleshot Linux service and connectivity issues

---

## Simulated Ticket Scenarios

- Password reset requests
- Account lockout resolution
- Missing drive troubleshooting
- Printer troubleshooting
- Software installation support
