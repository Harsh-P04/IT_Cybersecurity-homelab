# Network Security Analysis

## Objective

Perform network discovery, traffic inspection, and vulnerability assessments to simulate cybersecurity analysis and network security operations in an enterprise environment.

This phase focused on identifying devices, analyzing network communication, inspecting protocols, and performing vulnerability scanning using industry-standard security tools.

---

## Environment

| System | Role | IP Address |
|---------|------|-------------|
| Kali Linux | Security Testing | 192.168.10.40 |
| Windows Server 2022 | Domain Controller | 192.168.10.10 |
| Windows 11 | Domain Client | 192.168.10.20 |
| Ubuntu Server | Helpdesk + SIEM | 192.168.10.30 |

---

## Technologies Used

- Nmap
- Wireshark
- OpenVAS / Greenbone
- Kali Linux

---

## Tasks Performed

- Performed host discovery and network enumeration
- Conducted port scanning using Nmap
- Inspected network traffic using Wireshark
- Analyzed DNS, SMB, Kerberos, and LDAP traffic
- Performed vulnerability scanning using OpenVAS
- Identified exposed services and vulnerabilities
- Investigated network communication patterns

---

## Key Skills Demonstrated

- Network Enumeration
- Packet Analysis
- Vulnerability Assessment
- Traffic Inspection
- Threat Identification
- Protocol Analysis
- Cybersecurity Monitoring
