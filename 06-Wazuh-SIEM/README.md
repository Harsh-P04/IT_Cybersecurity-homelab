# Wazuh SIEM Monitoring

## Objective

Deploy and configure Wazuh SIEM to implement centralized logging, endpoint monitoring, threat hunting, and security event analysis in a SOC-style environment.

This phase focused on collecting endpoint telemetry, monitoring Windows activity, investigating security events, and performing threat hunting through a centralized dashboard.

---

## Environment

| System | Role | IP Address |
|---------|------|-------------|
| Ubuntu Server | Wazuh SIEM Server | 192.168.10.30 |
| Windows Server 2022 | Monitored Endpoint | 192.168.10.10 |
| Windows 11 | Monitored Endpoint | 192.168.10.20 |
